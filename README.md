# Enact Start

WebOS 에 사용되는 Enact를 이용하여 웹앱 개발을 바로 시작할 수 있는 스타터 킷 입니다.

Enact.js 관련 문서는 [여기](./docs/README.md) 를 참고해주세요.

## 기본 설치 패키지

아래와 같은 기본 설치 패키지가 제공됩니다.

- @enact/core
- @enact/i18n
- @enact/sandstone
- @enact/spotlight
- @enact/ui
- @enact/webos
- typescript
- styled-components

## 프로젝트 설정 방법

### 1. `package.json` 수정

- name, version, description 을 수정합니다.

### 2. `webos-meta/applinfo.json` 수정

- 진행하려는 프로젝트에 맞게 수정합니다.

### 3. `npm install` 실행

### 4. `npm run serve` 실행
